Intesa come le differenze con la gestione della reattività tramite il Virtual DOM

Gestione Reattività 🚀

Signal

fine-grained 🍚

React

coarse-grained 🪨
  • "fine-grained": i signals di Angular permettono di tracciare in modo automatico le dipendenze reattive. Quando il valore di un signal cambia, solo le parti del DOM o della logica che dipendono da quel valore vengono aggiornate.
  • "coarse-grained": ogni aggiornamento dello stato provoca il re-render dell'intero componente, e bisogna ottimizzare con useMemo, useCallback o React.memo per evitare calcoli eccessivi.
